Генератор паролей на Python
Простой консольный генератор паролей на Python, который:
•	генерирует случайный пароль нужной длины;
•	спрашивает URL сайта и логин/почту;
•	добавляет текущие дату и время;
•	сохраняет всё в текстовый файл `passwords.txt` в виде одной строки на запись.
Требования
•	Python 3.8+
•	macOS (но код будет работать и на Linux/Windows при установленном Python)
Установка
1.	Клонируй/скачай проект в любую папку (у тебя это `random-generator-password`).
2.	Убедись, что в папке есть файл `password_manager.py`.
Запуск
В терминале перейди в папку проекта:
cd /Users/ТВОЙ_ПУТЬ/Desktop/GIT/random-generator-password

Запусти скрипт:
python3 password_manager.py
